# ◎ Product Discovery & Strategy — Claude.ai Skill

> Runs structured product discovery and produces client-ready strategy artifacts. Built by [Observer Studio](https://observerstudio.co).

Part of the [Observer Studio Claude Skills collection](https://github.com/observer-studio/claude-skills).

---

## Install

```bash
npx claude-skill-product-discovery
```

Then upload the downloaded `.skill` file in **claude.ai → Settings → Skills**.

---

## What it produces

| Artifact | Description |
|---|---|
| Discovery Brief | Problem statement, target users, key assumptions, open questions |
| Research Plan | Methodology, participant criteria, discussion guide |
| Product Strategy Doc | Vision, business model, AI strategy, roadmap |
| Competitive Analysis | Landscape table, positioning map, whitespace |

Every output applies four lenses: MENA market context · AI-native thinking · Business model · Technical feasibility.
